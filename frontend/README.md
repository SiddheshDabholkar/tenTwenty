## Getting Started

Node Version
```
v22.20.0
```

1. Install dependencies
    ```bash
    npm install
    ```

2. Run the server
    ```bash
    npm run dev
    ```

3. Add ENV's
    ```bash
    REACT_APP_API_URL=http://localhost:3000
    ```

## This webiste is built using
1. Next js with app router
2. Tailwind css
3. Schadcn UI
4. axios
